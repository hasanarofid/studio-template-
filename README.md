# Studio Template - Modern Digital Product Website

A premium, modern SaaS-style website built to sell digital products (website templates). Designed for high conversion with a startup aesthetic, featuring clean typography, plenty of whitespace, and smooth interactions.

## 🚀 Features

- **Clean SaaS Aesthetic**: Modern minimal design with soft shadows and rounded corners.
- **Micro-interactions**: Scroll animations and visual feedback for better user engagement.
- **Conversion-Optimized Pricing**: 3-tier strategy (Starter, Business, Premium) with a highlighted "Main Product" and upsell services.
- **Responsive Design**: Optimized for everything from mobile phones to high-resolution desktops.
- **Fast Performance**: Built with zero dependencies using Vanilla HTML, CSS, and JS.
- **Pre-loaded Content**: Includes 3 template categories (Business, Portfolio, UMKM showcase).

## 🛠️ Built With

- **HTML5**: Semantic and SEO-friendly structure.
- **CSS3**: Premium design system with variables and responsive grids.
- **JavaScript**: Pure Vanilla JS for smooth animations and interactive components.
- **Vite**: Modern build tool for fast local development.

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS recommended)
- [npm](https://www.npmjs.com/) (included with Node.js)

### Installation

1. **Clone the repository**

   ```bash
   git clone git@github.com:hasanarofid/studio-template-.git
   cd studio-template-
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
